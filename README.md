## Description

A **car price prediction model** using a multi-brand used car dataset. Explores and compares multiple regression techniques to predict vehicle prices based on features like mileage, year, engine size, and fuel type.

The project includes full exploratory data analysis, feature engineering, model comparison, and a comprehensive report.

## Technologies

- **Language:** R
- **Libraries:** tidyverse, caret, randomForest
- **Output:** PDF report (R Markdown)

## Dataset

Multi-brand UK used car listings covering Audi, BMW, Ford, Hyundai, Mercedes, Skoda, Toyota, Vauxhall, and VW.
